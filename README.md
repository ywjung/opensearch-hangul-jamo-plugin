# Elasticsearch Hangul Jamo & Chosung Analyzer Plugin
* Hangul(Korean) Jamo, Chosung filter for an Elasticsearch
* [Korean version of README.md](README_KR.md) 

## Supported version
Below opensearch versions are tested
<br>
2.8.0

## Getting Started

### How to build
```
mvn clean package
```

### Installing
```
${opensearch.home}/bin/opensearch-plugin install file://${project.basedir}/target/releases/opensearch-hangul-jamo-analyzer-{version}.zip
```
